# Linux-Bash-Script
This is ITNH Linux Bash script that does some stuff.
to run, paste nano checklist.sh && chmod +x checklist.sh && alias checklist='sudo ./checklist.sh'
in the terminal. GNU Nano opens, and paste the script into nano, save, and exit. a new command is made called checklist, you can run it with checklist or use flags like --forensics. All flags are --harden(bypasses beggining warning), --forensic/--forensics(does not make system changes), --dry-run(runs with print statements tellign you what it would do.
